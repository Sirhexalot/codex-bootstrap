# Automatisierungen

Hier liegen konkrete Automatisierungen für den späteren Agenten.

Neue Automatisierungen sollen aus `.bootstrap/templates/automation-template/` kopiert und danach projektspezifisch angepasst werden.

Jede Automatisierung braucht mindestens:

- `README.md`
- `Runbook.md`
- `Memory.md`

Wichtig: Die jeweilige `Memory.md` muss bei jedem Lauf aktualisiert werden.
