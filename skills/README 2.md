# Projektspezifische Skills

Hier koennen projektspezifische Skills vorbereitet oder dokumentiert werden.

Neue Skills sollten aus `templates/skill-template/` kopiert werden, wenn eine Arbeitsweise mehrfach gebraucht wird.

Externe oder uebernommene Skill-Sammlungen koennen projektlokal unter `skills/upstream/` liegen. Direkte projektlokale Skills mit eigener `SKILL.md` liegen weiterhin direkt unter `skills/<skill-name>/`.

Ein guter Skill beschreibt:

- Wann er genutzt wird.
- Welche Eingaben er erwartet.
- Welche Schritte der Agent befolgen soll.
- Welche Ergebnisse entstehen sollen.
- Welche Fehler oder Grenzen zu beachten sind.
