# Automation Template

Diese Vorlage ist der Startpunkt für neue Codex-Automatisierungen. Kopiere den Ordner, benenne ihn passend zur Automatisierung um und passe die Platzhalter an.

## Zweck

Beschreibe hier kurz:

- Was die Automatisierung macht.
- Wann oder wodurch sie gestartet wird.
- Welche Ergebnisse sie erzeugt.
- Welche Personen, Teams oder Systeme betroffen sind.

## Voraussetzungen

- Benötigte Konten, Apps oder MCP-Server
- Benötigte lokale Dateien oder Ordner
- Benötigte Umgebungsvariablen

## Dateien

- `README.md`
- `Memory.md`
- `Runbook.md`
